# Screenshots para Google Play

Google Play requiere minimo 2 capturas de pantalla. Recomendado: 4-8.

## Dimensiones:
- Telefono: 1080x1920 (portrait) o 1920x1080 (landscape)
- Minimo: 320px, Maximo: 3840px

## Capturas recomendadas (en orden):
1. `start-screen.png` - Pantalla de inicio con el formulario
2. `questions-screen.png` - Respondiendo preguntas
3. `results-triangle.png` - Resultados con grafico triangular
4. `results-scores.png` - Puntuaciones y tipo de amor
5. `pdf-export.png` - Exportacion PDF

## Como tomarlas:
1. Ejecutar la app en un emulador o dispositivo
2. Usar la herramienta de captura de Android Studio o del dispositivo
3. Emulador recomendado: Pixel 7 (1080x2400)
